Flawta Acoustic Documentation

This project hosts the bilingual reference for Flawta, a system for documenting and classifying acoustic units of traditional metallic instruments. It includes a lexicon of sound units, a naming standard, and the methodology used for classification.

Standard

  • Core lexicon supports chain, bell, and grammar units.
  • IDs are canonical and map to audio/spectrogram assets.
  • Entries are expandable while preserving naming consistency.

Methodology

The documentation process follows a structured pipeline to ensure consistency and reproducibility across all lexical entries.

  1. Define lexical unit IDs and bilingual descriptors.
  2. Attach source audio samples.
  3. Attach spectrogram images with matching filenames.
  4. Iterate classifications as new validated units appear.
